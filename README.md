# Java-Polymorphism-and-Interfaces
Payroll system to include an additional Employee subclass like PieceWorker. 
This program expands an existing payroll system by introducing a new Employee subclass called PieceWorker. Unlike other employees, a PieceWorker is paid based on the number of pieces they produce. The class includes two private instance variables, wage (wage per piece) and pieces (number of pieces produced), and implements the earnings method to calculate total pay as wage Ã— pieces.

An array of Employee objects is then created to store different employee types (including PieceWorker). For each employee, the program displays both their string representation and their calculated earnings
